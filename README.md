# Installation du projet

Pour ouvrir ce projet, certaines conditions doivent être respectées :

- Installer **Unity Editor version 6000.2.11f1**  
  *(Ne pas tenir compte de l’alerte de sécurité affichée par Unity Hub.)*

- Télécharger cette version de Unity à partir de l’archive officielle :  
  https://unity.com/fr/releases/editor/archive

---

# Instructions pour tester le jeu

Pour tester correctement le jeu :

- Il est nécessaire d’utiliser un **casque de réalité virtuelle**, idéalement un **Meta Quest 3**.
- Le jeu **ne peut pas être testé complètement sans casque VR**.

Cependant, il est tout de même possible :

- d’explorer l’environnement
- de visualiser les scènes

directement dans **la vue Scene de l’éditeur Unity**.

# Volet programmation

Tous les scripts du projet se trouvent dans le dossier :

Etincelle/Assets/Scripts


## Scripts principaux (Managers)

Ces scripts contrôlent les systèmes principaux du projet :

- `GameManager.cs`
- `QuestManager.cs`
- `AudioManager.cs`
- `TimelineManager.cs`
- `UIManager.cs`

## Scripts de transformation de matériaux

Ces scripts gèrent certaines transformations visuelles dans l'environnement :

- `DynamisationMeuble.cs`
- `DisparitionVille.cs`

## Scripts des mécaniques d’interaction

Ces scripts contrôlent les interactions du joueur avec les objets :

- `OrdinateurTextInput.cs`
- `GrabDetection.cs`
- `ToucherDetection.cs`

## Scripts d’interface utilisateur (UI)

- `UIFollowHead.cs`
